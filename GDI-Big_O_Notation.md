# Big-O-Notation

## Hierarchie:  

- from fast  
- O( 1 )  
- O( log(n) )  
- O( n )  
- O( n*log(n) )  
- O( n^2 )  
- O( n^3 )  
- O( n^k ) // k > 3  
- O( k^n )  
- O( n! )  
- O( n^n )  
- O( infinite )  
- to slow  


    42log(5n)n² + 42log(5n)⁵n     = O( log(n)n² )  
    (65n + 36π²ⁿ)⁵                = O( π¹⁰ⁿ )  
    45log(5n)⁵n² + 42log(5n)n²    = O( log(n)⁵n² )

    (12/n⁶) + 794⁶n               = O( n )
